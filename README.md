# How It Works

1. The user enters the desired password length.
2. The program combines:
   - Uppercase letters
   - Lowercase letters
   - Numbers
   - Special characters
3. A random password is generated using these characters.
4. The user is asked whether they want to generate another password.
5. The program continues until the user chooses to exit.
